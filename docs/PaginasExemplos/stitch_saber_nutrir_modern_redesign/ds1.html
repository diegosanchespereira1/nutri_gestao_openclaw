Design System v1.0

Motion & Interaction

Sistema de animações premium para o redesign Saber Nutrir. Fade-in suave, reveal de imagens com scale cinematográfico e botões com shine sweep.

Motion Tokens

Easing Curves

  • --ds-ease-premium · cubic-bezier(0.16, 1, 0.3, 1) — entrada principal
  • --ds-ease-out-soft · cubic-bezier(0.22, 1, 0.36, 1) — saída suave
  • --ds-ease-spring · cubic-bezier(0.34, 1.56, 0.64, 1) — micro-interações
  • --ds-ease-in-out · cubic-bezier(0.65, 0, 0.35, 1) — transições longas

Durations

  • --ds-duration-instant · 200ms — hover/active
  • --ds-duration-fast · 450ms — botões, chips
  • --ds-duration-normal · 700ms — textos, cards
  • --ds-duration-slow · 1000ms — imagens
  • --ds-duration-cinematic · 1400ms — hero, ken-burns

Distances & Blur

  • --ds-translate-sm · 12px — botões
  • --ds-translate-md · 24px — textos
  • --ds-translate-lg · 40px — seções
  • --ds-blur-subtle · 6px — fade-up inicial
  • --ds-scale-from · 1.06 — image reveal

Stagger

  • --ds-stagger-step · 80ms entre elementos
  • Classes: .ds-delay-1 … .ds-delay-6
  • Atributo: data-ds-delay="N" (multiplica step)

Classes de Animação

.ds-fade-in

Opacidade 0 → 1. Uso: overlays, badges.

.ds-fade-up

Sobe 24px + blur sutil. Uso: títulos, parágrafos.

.ds-fade-scale

Scale 0.96 → 1. Uso: cards, stat blocks.

.ds-image-reveal

Container com overflow hidden. Imagem scale 1.06 → 1.

.ds-hero-image

Ken-burns automático no load. Uso: hero background.

.ds-btn

Fade-up + shine sweep no hover. Modificadores: primary, secondary.

.ds-text-link

Fade-up para links de texto. Ícone desliza no hover.

Ativação: adicionar classe is-visible via Intersection Observer (data-ds-animate) ou imediatamente no hero com data-ds-immediate.

Como usar

<div class="ds-image-reveal rounded-xl" data-ds-animate>
  <img src="..." alt="..." />
</div>

<!-- Texto com fade-up -->
<h2 class="ds-animate ds-fade-up" data-ds-animate>Título</h2>

<!-- Script no final do body -->
